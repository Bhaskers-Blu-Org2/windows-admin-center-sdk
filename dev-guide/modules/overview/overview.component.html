Overview

Welcome to Windows Admin Center Developer Tools!

In this tool set, you will find development examples and scenarios to help you build an extension in Windows Admin Center. With these examples, you can interact with them in runtime, as well explore the included source code in the SDK. Be sure to check out documentation included in the SDK, which has step-by-step instructions and additional information to help you get an extension up and running in Windows Admin Center.

Development Guide

The Development Guide tool provides an in-depth look at all the individual controls, styles, pipes and also the SME shell services that are available in Windows Admin Center, and you can find examples of how to use each tool in the source code, there are also bunch of code examples.

Developer Tools include the following sections:

  • Injected Styles
  • Angular Controls
  • Angular Pipes
  • MsftSme Utilities
  • Accessibility
  • Shell Services
  • Gateway
